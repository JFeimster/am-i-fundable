# Funding Examples

This folder contains public-safe example assets for the Funding Readiness Scorecard ecosystem.

## Files

- `funding-kb.html` — static knowledge-base style page explaining funding readiness concepts without private partner routing, rates, approvals, or sensitive data.

## Rules

These examples are safe for demos, docs, and public education.

Do not add:

- Real borrower records
- Private partner URLs
- Affiliate portal links
- Commissions
- Credentials
- Underwriting rules
- Approval or guaranteed funding claims

## Recommended Use

Use these examples to:

- Show what public funding education content can look like
- Give Jules/Codex/Cursor a safe static-page reference
- Seed future knowledge-base pages
- Support Custom GPT knowledge files with public-safe language

## Language Standard

Use readiness language, not approval language.

Good:

- “Possible fit”
- “Review path”
- “Prep checklist”
- “Common readiness signals”
- “Next step”

Avoid:

- “Approved”
- “Guaranteed”
- “Eligible”
- “Best rate”
- “Underwritten”
- “No risk”
