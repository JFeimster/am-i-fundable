# Knowledge Base

This directory contains resources meant to support Custom GPT knowledge uploads, repository navigation, and internal operating context for the Funding Readiness Scorecard.

## Structure

- `/funding` - Context about funding products, readiness, and routing concepts.
- `/automation` - Context on workflow automations and integrations.
- `/crm` - Best practices for passing scorecard data to a CRM.
- `/engineering-as-marketing` - How to leverage the scorecard as a growth and lead-capture tool.

## Guidelines

- **Compliance Safe:** All documents must adhere strictly to compliance guardrails (no promises of approval, no guaranteed outcomes).
- **Generic:** Use placeholders for provider names and CRM values (e.g., `Demo Provider`, `REDACTED_FOR_PUBLIC_REPO`).
- **Educational:** Documents here explain *how* the system works operationally without exposing sensitive routing data.
