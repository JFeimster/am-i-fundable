# Documentation

This folder contains the operational documentation for deploying, testing, and managing the Funding Readiness Scorecard.

## Contents

- `compliance-guardrails.md`: Explains the strict compliance rules for language and data handling.
- `embed-instructions.md`: How to embed the scorecard widget.
- `deployment.md`: Instructions for deploying to Vercel.
- `vercel-env.md`: Environment variables required in Vercel.
- `testing-checklist.md`: Steps for manual and automated testing.
- `merge-readiness-checklist.md`: Steps to verify before merging changes.
- `naming-conventions.md`: Standards for file and variable names.
